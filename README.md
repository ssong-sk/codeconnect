### 팀 멤버
- ### 🐻김태윤 
  - GitHub 주소 : [[GitHub](https://github.com/dancua)]
  - 담당 역할 : 메인페이지,마이페이지,로그인 회원가입 디자인 및 기능
  - 상세 기능 : 회원가입 : 사용자의 정보를 양식에 맞게 입력할 수 있도록 유효성 검사 진행 후 입력한 정보를 DB에 저장하는 기능
  - 로그인 : 저장된 사용자 정보를 불러와 웹 사이트의 서비스를 이용 가능하게 도와주는 기능
  - 메인페이지 : 웹사이트 소개, 회사 소개, 마이페이지로 이동할 수 있는 메인페이지
  - 마이페이지 : 회원의 정보를 수정 탈퇴, 관심있는 회사 북마크, 회원이 작성한 이력서를 수정이 가능하며 지원한 회사의 현황을 확인할 수 있는 마이페이지
